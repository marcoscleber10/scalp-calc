📘 Estratégia 1: Market Making Adaptativo

🎯 Objetivo

Detectar fluxo agressivo de compra/venda e ajustar o range da posição de liquidez ANTES do preço se mover.

⚙️ Como funciona (exemplo prático)

Cenário: Posição LP em SOL/USDC com range de ±10% em torno de $150.

  • Volume comprador > 65% → range para cima ([145-175])
  • Volume vendedor > 65% → range para baixo ([135-165])

📊 Exemplo numérico

Preço $150, volume comprador 72% → novo range [147, 157.5]

⚠️ Riscos

Fluxo reverso pode causar perdas. Use dados confiáveis.

📘 Estratégia 2: Gamma Scalping

🎯 Objetivo

Usar LP como posição delta neutra e hedge via spot.

⚙️ Como funciona

ETH/USDC: se sobe → vende spot (realiza lucro); se cai → compra spot.

📊 Exemplo

Capital $10.000, ETH $2.000 → 5 ETH. Sobe para $2.100 → vende 0,5 ETH ($1.050), lucro realizado.

📘 Estratégia 3: Liquidity Sniping

🎯 Objetivo

Entrar cedo em pools novas com APR alto.

⚙️ Como funciona

Detecta pool nova + APR > 200% → aloca capital parcial. Sai se APR cair rápido.

📊 Exemplo

Pool XYZ/USDC com 550% → aloca 10% ($1.000). Após 2 dias, APR 80% → sai com lucro.

🤖 Bot de Liquidez Automatizado (Solana - Meteora / Orca)

Este bot gerencia automaticamente uma posição de liquidez em um pool CLMM na Solana. Ele ajusta o range de preço conforme a volatilidade e protege o capital em caso de queda brusca.

📌 Funcionalidades

  • 🔵 Adicionar liquidez – Cria uma posição com range de ±10% em torno do preço atual.
  • ⚡ Auto-ajuste – Se o preço variar mais de 5%, o range é reposicionado para o novo preço.
  • 🚨 Modo Crash – Se o preço cair mais de 8%, toda a liquidez é removida para evitar perdas.
  • 📊 Preço em tempo real – Busca o preço da SOL na Binance a cada 20 segundos.

⚙️ Como usar (para desenvolvedores)

  1. Instale as dependências: npm install @solana/web3.js axios dotenv @orca-so/whirlpools-sdk
  2. Crie um arquivo .env com sua chave privada: PRIVATE_KEY=...
  3. Altere o endereço da pool (POOL_ADDRESS) para o pool desejado (ex: SOL/USDC da Meteora).
  4. Execute com node index.js

📄 Código completo (JavaScript)

// =============================================\n// METEORA CLMM BOT (REAL LP + AUTO RANGE)\n// =============================================\nrequire('dotenv').config()\nconst { Connection, Keypair, PublicKey } = require('@solana/web3.js')\nconst axios = require('axios')\nconst { WhirlpoolContext, buildWhirlpoolClient, ORCA_WHIRLPOOL_PROGRAM_ID } = require("@orca-so/whirlpools-sdk")\n\nconst RPC = "https://api.mainnet-beta.solana.com"\nconst connection = new Connection(RPC, "confirmed")\nconst secret = JSON.parse(process.env.PRIVATE_KEY)\nconst wallet = Keypair.fromSecretKey(Uint8Array.from(secret))\nconsole.log("Wallet:", wallet.publicKey.toString())\n\nconst ctx = WhirlpoolContext.withConnection(connection, wallet, ORCA_WHIRLPOOL_PROGRAM_ID)\nconst client = buildWhirlpoolClient(ctx)\n\nconst POOL_ADDRESS = new PublicKey("HJPjoWUrhoZzkzZbXj4iAFH4x6U7kK...")\n\nasync function getPrice() {\n    const res = await axios.get("https://api.binance.com/api/v3/ticker/price?symbol=SOLUSDT")\n    return parseFloat(res.data.price)\n}\n\nfunction priceToTick(price) {\n    return Math.floor(Math.log(price) / Math.log(1.0001))\n}\n\nasync function addLiquidity(rangePercent = 10) {\n    const whirlpool = await client.getPool(POOL_ADDRESS)\n    const price = await getPrice()\n    const lower = price * (1 - rangePercent / 100)\n    const upper = price * (1 + rangePercent / 100)\n    const tickLower = priceToTick(lower)\n    const tickUpper = priceToTick(upper)\n    console.log("🔵 ADD LP")\n    console.log("Range:", lower, "-", upper)\n    const liquidityAmount = 1000000\n    const tx = await whirlpool.openPosition(tickLower, tickUpper, liquidityAmount)\n    const sig = await tx.buildAndExecute()\n    console.log("✅ Liquidez adicionada:", sig)\n}\n\nasync function removeLiquidity(position) {\n    console.log("🔴 Removendo LP...")\n    const tx = await position.closePosition()\n    const sig = await tx.buildAndExecute()\n    console.log("❌ Liquidez removida:", sig)\n}\n\nlet currentPosition = null\nlet lastPrice = 0\n\nasync function managePosition() {\n    const price = await getPrice()\n    if (!lastPrice) { lastPrice = price; return }\n    const change = ((price - lastPrice) / lastPrice) * 100\n    console.log("Preço:", price, "| Δ:", change.toFixed(2), "%")\n    if (change >= 5 || change <= -5) {\n        console.log("⚡ Reposicionando range...")\n        if (currentPosition) await removeLiquidity(currentPosition)\n        await addLiquidity(10)\n    }\n    if (change <= -8) {\n        console.log("🚨 CRASH - SAINDO")\n        if (currentPosition) await removeLiquidity(currentPosition)\n    }\n    lastPrice = price\n}\n\nsetInterval(managePosition, 20000)\n\n// INSTRUÇÕES: node index.js
